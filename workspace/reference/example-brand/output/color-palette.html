04

Usage Guidelines

When to reach for each color. Every application should be intentional — if you're choosing a color because it "looks nice," step back and ask what it needs to communicate.

Navy
Use for: Primary text, page headers, navigation bars, hero backgrounds, key UI surfaces, document titles. Navy is the default text color — never use pure black (#000000).

Avoid: Small decorative elements where it reads as generic dark. Navy earns its weight through scale.
Teal
Use for: Call-to-action buttons, hyperlinks, active states, data highlights, progress bars, selected tabs, accent borders. Teal is the action color — it means "go."

Avoid: Large background fills or body text. Teal should punctuate, not dominate.
Slate
Use for: Secondary text, captions, metadata labels, subtle borders, placeholder text, input field labels. Slate is the supporting voice in the hierarchy.

Avoid: Headlines or primary UI text. Slate should support, not lead.
Coral
Use for: Alert badges, key metrics, important callouts, featured statistics, warning states. Coral speaks only when something demands attention.

Avoid: Background fills. Large coral surfaces become visually aggressive and undermine the calm authority of the palette.
White
Use for: Primary page backgrounds, content surfaces, compliance documents, report bodies. White is the default canvas — transparency made literal.

Avoid: Using off-whites or blue-tinted whites. The palette has Warm Gray for softer surfaces. White means white.
Warm Gray
Use for: Card backgrounds, section fills, alternate table rows, sidebar surfaces, input backgrounds. The differentiation layer that adds depth without color.

Avoid: Combining with off-whites that clash with the warm undertone. Warm Gray sits alone in its register.

The palette of earned trust.

Navy carries the weight of every engagement that went right. Teal marks the path forward. Coral speaks up only when something actually matters.

The palette lives at the intersection of institutional credibility and startup speed. It's what financial services looks like when it's built for companies that ship weekly. Precise enough for an audit report. Modern enough for a product dashboard. The colors don't perform authority — they've earned it.
