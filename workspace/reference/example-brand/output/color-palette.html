01

Base Colors

Six foundational colors define the Baseline visual identity. Navy anchors the brand with authority and trust. Teal drives action and draws the eye. Slate provides quiet structure. Coral speaks up when something matters. White and Warm Gray create breathing room for content to do its work.

02

Tint Scales

Ten-step tint scales for each chromatic color, generated from HSL lightness adjustments. The 500 step is the base color. Lighter values (50–400) work well for backgrounds, borders, and hover states. Darker values (600–900) provide depth for text, shadows, and high-contrast surfaces.

03

Color Pairings

Recommended foreground/background combinations with WCAG 2.1 contrast ratios. Every primary text pairing must meet AA (4.5 : 1) or AAA (7 : 1) for normal text. Decorative and large-text uses may accept AA Large (3 : 1).
