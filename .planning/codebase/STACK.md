**Build Process:**
1. TypeScript compilation: `tsc -b`
2. Vite bundling and minification
3. Tailwind CSS generation with theme tokens
4. Static assets (fonts, SVGs) bundled

**Development Server:**
- Vite dev server on port 3001
- Hot Module Replacement (HMR) for instant feedback
- `serveWorkspace` plugin serves `/workspace/` files for dynamic content loading during development

**Static Content Loading:**

- No runtime file I/O or HTTP requests to load content
