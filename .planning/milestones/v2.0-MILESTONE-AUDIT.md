### content-loader.ts Token File Format Gap

The design-kit glob `*.{html,md}` excludes `.css`, `.js`, and `.json` files. Token files cannot be returned by `getFile()` or included in file lists. However, Vite serves token CSS directly for iframe rendering via `<link>` tags, so HTML specimens render correctly. Impact is limited to dashboard file listing/counting.
