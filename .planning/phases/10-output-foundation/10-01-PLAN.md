**### Task 1: Token Extraction**

Instructions for the agent to:
1. Read `workspace/drafts/visual-direction.md`
2. Extract the CSS Custom Properties Block from the `## Design System Parameters` section
3. Parse properties into three categories: colors (--color-*), typography (--font-*, --font-weight-*, --letter-spacing-*, --line-height-*), spacing (--radius-*, --shadow-*, --space-*, --content-max-width, --transition-speed)
4. Write 5 files to `workspace/output/design-kit/tokens/`:

**colors.css** -- `:root { }` block containing all `--color-*` properties. Include both palette colors and semantic tokens. Format:
```css
/* Design Tokens: Colors — extracted from visual-direction.md */
:root {
  /* Palette */
  --color-primary: #VALUE;
  ...
  /* Semantic */
  --color-text-primary: var(--color-primary);
  ...
}
```

**typography.css** -- `:root { }` block containing `--font-*`, `--font-weight-*`, `--letter-spacing-*`, `--line-height-*` properties. Format:
```css
/* Design Tokens: Typography — extracted from visual-direction.md */
:root {
  --font-display: 'FONTNAME', system-ui, sans-serif;
  ...
}
```

**spacing.css** -- `:root { }` block containing `--radius-*`, `--shadow-*`, `--space-*`, `--content-max-width`, `--transition-speed`. Format:
```css
/* Design Tokens: Spacing & Structure — extracted from visual-direction.md */
:root {
  --radius-sm: VALUE;
  ...
}
```

**tokens.json** -- All tokens in DTCG (Design Tokens Community Group) format. Structure:
```json
{
  "color": {
    "primary": { "$value": "#HEX", "$type": "color", "$description": "Primary brand color" },
    ...
  },
  "size": {
    "radius": { "sm": { "$value": "4px", "$type": "dimension" }, ... },
    "spacing": { "xs": { "$value": "0.25rem", "$type": "dimension" }, ... }
  },
  "shadow": { "sm": { "$value": "...", "$type": "shadow" }, ... },
  "fontWeight": { "heading": { "$value": "700", "$type": "fontWeight" }, ... },
  "fontFamily": { "display": { "$value": "...", "$type": "fontFamily" }, ... },
  "letterSpacing": { "heading": { "$value": "-0.01em", "$type": "dimension" }, ... },
  "lineHeight": { "body": { "$value": "1.7", "$type": "number" }, ... }
}
```

**tailwind.config.js** -- Tailwind v3 theme.extend mapping CSS var references:
```javascript
/** @type {import('tailwindcss').Config} */
module.exports = {
  theme: {
    extend: {
      colors: {
        'brand-primary': 'var(--color-primary)',
        'brand-accent': 'var(--color-accent)',
        ...
      },
      borderRadius: {
        'brand-sm': 'var(--radius-sm)',
        ...
      },
      spacing: {
        'brand-sm': 'var(--space-sm)',
        ...
      },
    },
  },
}
```

Include instruction: "Read the actual property names from visual-direction.md -- the names above are examples from the reference brand. Map whatever properties exist in the client's visual direction."

**### Task 2: HTML Post-Processing**

Instructions for the agent to:
1. Use Glob to find all `.html` files in `workspace/output/client/`
2. For each HTML file, read the content
3. Extract the `:root { ... }` block using regex. The block is always inside the first `<style>` tag, preceded by a comment containing "Brand Tokens" or similar. Use this regex approach:
   - Primary regex: `/\/\*[^\n]*Brand Tokens[^\n]*\*\/\s*:root\s*\{[^}]*\}/s`
   - Fallback regex if primary doesn't match: `/:root\s*\{([^}]*)\}/s` (match the first `:root` block only -- do NOT use global flag)
4. Replace the matched `:root` block with a comment: `/* Tokens loaded from external files */`
5. Insert `<link>` tags before `</head>`:
   ```html
   <link rel="stylesheet" href="../tokens/colors.css">
   <link rel="stylesheet" href="../tokens/typography.css">
   <link rel="stylesheet" href="../tokens/spacing.css">
   ```
6. Write the modified HTML to `workspace/output/design-kit/[same-filename]`
7. Do NOT modify the original file in `workspace/output/client/` -- the client specimens remain self-contained (per D-07, D-08)

Include warning: "The regex must target only the FIRST `:root {}` block. ui-kit.html may contain component-level custom property declarations -- do not strip those."
