# SRC-021: One-Page Brand Canvases — BrandScript, Brand Key, Brand Pyramid

**Source type:** Framework methodology / Industry tools
**Credibility:** High (StoryBrand official workbook, Unilever Brand Key via Beloved Brands and Toolshero, NNGroup, industry standard frameworks)
**Date accessed:** 2026-03-26
**Sources:** Umbrex SB7 framework guide (umbrex.com), StoryBrand workbook PDF (storybrand.com), Agency Boon BrandScript template (agencyboon.com), Beloved Brands Brand Key model (beloved-brands.com/brand-key-model), Toolshero Brand Key explanation (toolshero.com), Smartsheet Brand Pyramid templates (smartsheet.com), Iterators Brand Pyramid framework (iteratorshq.com), Gravity Global StoryBrand guide (gravityglobal.com)

## Key Findings

### StoryBrand BrandScript — One-Page Narrative Summary

The BrandScript is a one-page document with 7 elements:
1. **Character** — who the customer is and what they want
2. **Problem** — external, internal, and philosophical expressions
3. **Guide** — empathy + authority proof (the brand)
4. **Plan** — simple 3-step or rules-based path
5. **Call to Action** — direct and transitional offers
6. **Failure** — risks of inaction
7. **Success** — tangible outcomes and emotional payoff

Deliverable chain from BrandScript:
- BrandScript → One-liner (single sentence: Character + Problem + Plan + Success)
- BrandScript → Website wireframe (map SB7 to page sections)
- BrandScript → Channel-native copy

Key insight: "Teams often codify the seven parts into a one-page 'BrandScript,' then use it to drive a website wireframe, one-liner, and channel-native copy. The core logic is modular: not every asset needs all seven parts, but the overall program should reinforce the full arc."

### Unilever Brand Key — One-Page Strategic Summary

The Brand Key model originated at Unilever. Nine elements:
1. **Root Strengths** — brand's historical foundation
2. **Competitive Environment** — who the brand competes with
3. **Target** — primary audience
4. **Insight** — key customer insight that drives the brand
5. **Benefits** — functional and psychosocial
6. **Values & Personality** — what the brand stands for, how it behaves
7. **Reasons to Believe** — credibility arguments
8. **Discriminator** — "only brand X has..."
9. **Brand Essence** — 2-3 word summary

Purpose: "lay out the unique selling proposition (USP) elements of their brand on one page."

### Brand Pyramid — One-Page Hierarchical Summary

Five tiers (bottom to top):
1. **Features & Attributes** — product/service description and value
2. **Functional Benefits** — why customers use it, expected problem-solving
3. **Emotional Benefits** — mood/feelings after use
4. **Brand/Product Persona** — characteristics and values as a person
5. **Brand Idea/Essence** — core differentiator and value proposition

### What These Share

All three are **one-page synthesis artifacts** that:
- Force prioritization (can't fit everything)
- Create alignment (entire team references same page)
- Drive downstream execution (copy, design, decisions all derive from it)
- Are living documents (refined over time based on market feedback)

### Relevance to Brand Compass

Brand Compass produces comprehensive documents (brand foundation, voice guide, visual system, toolkit) but no one-page synthesis. Phase 2 benchmarking (SRC-002, SRC-004) already identified this gap:
- StoryBrand BrandScript provides customer-hero narrative summary
- GV Brand Sprint produces a "cheat sheet" for alignment
- Brand Key model provides strategic USP summary

A Brand Compass one-page canvas could synthesize:
- Core Belief (Phase 1) → Brand Essence
- Audience (Phase 2) → Target + Insight
- Positioning (Phase 3) → Discriminator + Competitive Environment
- Personality (Phase 4) → Values & Personality
- Tagline (Phase 5) → One-liner
- Voice tags (Phase 6) → Communication guide
- Color + Type (Phase 7) → Visual shorthand (hex codes, font stack)

This would be a new deliverable: a one-page **Brand Compass Card** that serves the same function as BrandScript/Brand Key/GV cheat sheet.

## Findings Tags

- **SUPPORTED** — One-page brand synthesis is a standard deliverable across multiple established frameworks (BrandScript, Brand Key, Brand Pyramid, GV cheat sheet)

- **SUPPORTED** — Customer-hero narrative framework (BrandScript) still absent from Brand Compass; this is the most significant messaging gap (confirmed Phase 4)
